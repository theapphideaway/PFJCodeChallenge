# PFJCodeChallenge

This project is a takehome challenge for Pilot/Flying J Android development. This is my implementation from the template that was given.

### Architecture/Libraries 
* MVVM
* Hilt
* Coroutines
* Live Data
* Retrofit
* OkHttp
* Kotlinx
* Androidx
* GoogleMaps
